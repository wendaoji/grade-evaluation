# 人员任职资格评价 APP 文档总览

本项目是一套面向企业内部任职资格评价场景的 Web 应用，支持管理员配置评价体系，评委对人员逐项评分，并基于加权得分率自动计算能力等级。

## 当前版本范围

- 支持 `管理员`、`评委`、`员工`、`主管`、`审计员` 五类角色
- 支持评价批次管理、状态切换与体系冻结
- 支持等级、维度、分类、评分项、权重、关键项门槛配置
- 支持按 `0 / 1 / 3` 评分、加权得分率计算与自动定级
- 支持等级分布、维度平均得分率、部门/岗位统计、个人结果列表、多批次趋势分析
- 支持批量导入人员、导入评价体系、应用规则模板、结果导出
- 支持员工自评、主管复核、多评委独立评分后汇总
- 支持审计日志、轻量 SSO 登录、健康检查、监控快照与数据库备份

## 当前实现形态

- 后端：原生 Node.js HTTP 服务
- 前端：原生 HTML/CSS/JavaScript 响应式页面
- 数据存储：SQLite，首次从 JSON 种子导入
- 目标定位：业务规则验证用 MVP
