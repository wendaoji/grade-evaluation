# 后续任务路线图

## 阶段 1：当前 MVP 已完成

- 单体应用可运行
- 支持管理员和评委角色
- 支持评价体系配置
- 支持加权评分和关键项门槛
- 支持自动定级和基础统计

## 阶段 2：工程化增强

当前已完成：

- SQLite 数据库存储替换 JSON 运行时存储
- PostgreSQL 建表脚本与迁移资产
- SQL 数据访问层已在 `src/store.js` 中收敛
- 首次启动自动导入 JSON 种子数据
- 密码哈希存储
- 输入校验和统一错误处理
- 基础分页、筛选和排序接口
- Docker / Docker Compose 单容器部署

## 阶段 3：业务能力增强

当前已完成：

- 支持部门和岗位维度统计
- 支持批量导入人员和评价体系
- 支持结果导出
- 支持多评价批次趋势分析
- 支持关键等级规则模板化

## 阶段 4：流程能力增强

当前已完成：

- 支持员工自评
- 支持主管复核
- 支持多评委独立评分后汇总
- 支持审批流与状态流转

## 阶段 5：企业化能力增强

当前已完成：

- 接入轻量单点登录映射
- 增加审计日志
- 增加操作留痕和变更历史
- 增加更细粒度的权限控制
- 增加部署、监控和备份方案
